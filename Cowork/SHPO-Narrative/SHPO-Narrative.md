# SHPO Narrative — Session Index

## How to use this project
At the start of every session:
1. Read `_memory.md` — standing instructions, Standards reference, project log, patterns
2. Read all files in `_inputs/` — HPE report, photos, scope of work, prior application
3. Draft narrative per standing instructions in `_memory.md`
4. Save output to `_outputs/` using the naming convention
5. Append one line to the Project Log in `_memory.md`
6. Append any SHPO reviewer feedback or successful language to Learned Patterns in `_memory.md`

## Folder structure
- `_inputs/` — HPE reports, photos, scope documents, prior SHPO correspondence
- `_outputs/` — Part 2 narratives, Section 106 narratives, SHPO letters
- `_memory.md` — running project memory, Standards reference, project log, patterns

# SHPO Narrative — Inputs

Drop raw files here before each session:
- Historic Preservation Evaluation (HPE) report
- Photographs (existing conditions)
- Scope of work description
- Prior SHPO correspondence or Part 1 approval
- Historic district nomination (if applicable)

Files are processed in `/scqa` → `/shpo` → `/edit-narrative` and moved to `_outputs/` when complete.

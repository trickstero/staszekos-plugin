# Zoning Analysis — Session Index

## How to use this project
At the start of every session:
1. Read `_memory.md` — standing instructions, jurisdiction triggers, case log, learned patterns
2. Read all files in `_inputs/` — these are the raw materials for this session
3. Perform analysis per standing instructions in `_memory.md`
4. Save output to `_outputs/` using the naming convention
5. Append one line to the Case Log in `_memory.md`
6. Append any new patterns learned to the Learned Patterns section in `_memory.md`

## Folder structure
- `_inputs/` — raw files dropped by Staszek before each session
- `_outputs/` — completed memos and design supplements
- `_memory.md` — running project memory, instructions, case log, patterns

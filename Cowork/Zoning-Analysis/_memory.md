# Zoning Analysis — Project Memory

## Standing Instructions
- Web search every analysis — never rely on memory for code language
- Research sequence: zoning GIS map → Municode/official code → overlays → FEMA flood → variance precedent → IBC check
- Output format: zoning-template.md structure — tables for dimensional standards, code citation on every standard, every non-conformity labeled with relief type
- Follow with design-supplement.md: massing calc, entitlement risk matrix, IBC flags, HTC flag if historic overlay

## Jurisdiction Triggers
- Chicago: ARO (10+ units), TSL (transit-served locations), Pedestrian Streets, Lakefront Overlay
- Illinois non-Chicago: IDOT access permit if state route frontage, IHPA for SHPO
- Idaho (Kingswood/Brekenwood): ACHD access (Ada County), ITD (state highway), subdivision plat sequence
- DC: DCSHPO + HPRB, Green Area Ratio, Inclusionary Zoning
- National: IBC occupancy/construction type matrix, ADA, NFPA 13/72

## Chicago PD Multi-Building FAR Bundling
When a PD site contains existing + new buildings:
- New building available FAR = total PD allowable − existing building consumed GSF
- Confirmed: 741 N. Wells (PD 1519, residential — 755 N. Wells consumed 15,996 SF)
- Confirmed: 609 W. Randolph (commercial — 601 W. Randolph consumed 13,843 SF)
- Always coordinate with land use counsel before presenting to client

## Chicago PD Trigger Thresholds
- >80 ft height OR >2 acres OR ≥75 DU → Planned Development required (§17-8-0400)
- Any change of occupancy or FAR addition on existing PD site → DPD minor change petition first

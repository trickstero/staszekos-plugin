# RFI Drafting — Session Index

## How to use this project
At the start of every session:
1. Read `_memory.md` — standing instructions, project registry, RFI log, patterns
2. Read all files in `_inputs/` — drawings, specs, contract references, field reports
3. Draft RFI per standing instructions in `_memory.md`
4. Save output to `_outputs/` using the naming convention
5. Append one line to the RFI Log in `_memory.md`
6. Append any communication preference notes to Learned Patterns in `_memory.md`

## Folder structure
- `_inputs/` — drawings, spec excerpts, contract clauses, GC correspondence
- `_outputs/` — Procore-ready RFI drafts
- `_memory.md` — running project memory, project registry, RFI log, patterns

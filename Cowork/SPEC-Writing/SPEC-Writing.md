# SPEC Writing — Session Index

## How to use this project
At the start of every session:
1. Read `_memory.md` — standing instructions, preferred manufacturers, section log, patterns
2. Read all files in `_inputs/` — product data, previous sections, owner requirements
3. Draft spec section per standing instructions in `_memory.md`
4. Save output to `_outputs/` using the naming convention
5. Append one line to the Section Log in `_memory.md`
6. Append any new preferred manufacturers or firm standards to `_memory.md`

## Folder structure
- `_inputs/` — product data sheets, cut sheets, substitution requests, owner lists
- `_outputs/` — completed 3-part MasterFormat spec sections
- `_memory.md` — running project memory, preferred manufacturers, section log, patterns

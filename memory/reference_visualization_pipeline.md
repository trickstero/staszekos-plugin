## Krea NB2 (Nano Banana 2 / Gemini)

**Opening line (always):** "Real photograph NOT a render NOT CGI"
**Camera specs to include:** Canon EOS R5, specific lens (85mm f/1.4 or 24-70mm f/2.8), ISO, shutter speed
**Prompt length limit:** Under 2,000 characters
**Thinking mode:** OFF by default. ON only for complex infographics.
**Workflow:** Batch API (50% discount) → generate at 512px → select best → upscale to 1K/2K/4K

## Todd Formula (Exterior Architecture)

Photorealistic, camera-language prompts for exterior architectural visualization.
- Describe as a photograph, not a design intent
- Specify time of day, weather, season
- Include foreground context (street, pedestrians, vehicles)
- Camera position: eye level or slight elevation, never bird's eye for hero shots

## JJ Style (Watercolor + Ink)

Watercolor + ink linework aesthetic.
- Delicate tree canopies, soft foliage
- Warm purple-mauve shadows
- Ink linework on building edges, lighter wash for surroundings
- Use for context diagrams, presentation boards, urban design exhibits
